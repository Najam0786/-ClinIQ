# Heart Disease Data

Place your heart disease CSV files here.

## Supported Dataset Types
| Type | Target Column | Example Values |
|---|---|---|
| Diagnosis | `target` / `condition` / `heart_disease` | 0 / 1 |
| Mortality | `death` / `deceased` | 0 / 1 |
| Risk | `high_risk` | 0 / 1 |

## Known Compatible Datasets (Kaggle)
- **Heart Disease UCI** — 13 clinical features, target: `target` (0/1)
- **Cleveland Heart Disease** — target: `condition` (0/1)
- **Heart Failure Clinical Records** — target: `DEATH_EVENT` (0/1)

## Key Features ClinIQ Recognises
`age`, `sex`, `cp` (chest pain type), `trestbps` (resting BP),
`chol` (cholesterol), `fbs` (fasting blood sugar), `restecg`,
`thalach` (max heart rate), `exang` (exercise angina),
`oldpeak`, `slope`, `ca` (vessels), `thal`
