# Stroke Data

Place your stroke CSV files here.

## Supported Dataset Types
| Type | Target Column | Example Values |
|---|---|---|
| Occurrence | `stroke` | 0 / 1 |
| Severity | `severe_stroke` | 0 / 1 |
| Mortality | `death` / `died` | 0 / 1 |

## Known Compatible Datasets (Kaggle)
- **Stroke Prediction Dataset** — 11 clinical + lifestyle features, target: `stroke` (0/1)

## Key Features ClinIQ Recognises
`age`, `gender`, `hypertension`, `heart_disease`, `ever_married`,
`work_type`, `Residence_type`, `avg_glucose_level`, `bmi`,
`smoking_status`
